Light [DRAFT]

A device that emits light. Typically it can be set to a range of colours and brightnesses, as well as some more advanced features such as alerts.

Overview

Device Type light
Data Type serialised JSON
Example Modules
  • Hue
More information
  • HSL and HSV
  • xyY (CIE 1931 colour space)
  • Colour temperature
Parameters
on
boolean
Optional		 Whether to power the light on or not. `true` is on, `false` is off.
Example: true
bri
number
Optional		 The brightness, measured as an 8bit unsigned integer (ie between 0 and 254).
Example: 100
hue
number
Optional		 The color measured as a 16bit unsigned integer (ie between 0 and 65535).
Example: 12000
sat
number
Optional		 Saturation, measured as an 8bit unsigned integer (ie between 0 and 254).
Example: 10
xy
array
Optional		 The colour in CIE 1931 colour space represented as an array of x,y co-ordinates.
Example: [0.4,0.6]
ct
number
Optional		 Colour temperature (luminance) between 154 (cold) and 500 (warm)
Example: 154
alert
number
Optional		 Flash the light this many times (between 1 and 10).
Example: 10
transitionTime
number
Optional		 The time to transition to the new state in miliseconds.
Example: 2000
